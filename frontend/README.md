# LitXplore Next.js Application

This directory contains the complete LitXplore application: React pages, Next.js API route handlers, authentication, database access, AI features, PDF handling, and document export.

```bash
cp .env.example .env.local
npm install
npm run dev
```

Use Node.js 20 or newer. See the repository root `README.md` for environment variables and Vercel deployment instructions.
